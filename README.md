# 每日游戏开发学习日志



这个仓库记录我从 2026-06-08 开始，每天学习游戏开发（算法、数学、引擎、C++、Shader 等）的代码、笔记和项目。



## 目录结构



* `algorithms/` - 数据结构与算法练习（排序、查找、动态规划等）
* `math/` - 数学与图形学笔记和Demo（向量、矩阵、四元数等）
* `engine/` - 引擎学习（Unity/UE 的小项目或测试场景）
* `cpp/` - C++ 语法、特性、内存管理等练习
* `shaders/` - 练习 （顶点着色器、片元着色器等）
* `project/` - 综合小项目（比如一个小游戏原型）
