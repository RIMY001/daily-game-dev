\# Git/GitHub 学习笔记



\## 2026-06-08



\### 今天学到的命令

\- `git clone <url>` – 克隆远程仓库到本地

\- `git add .` – 暂存所有改动

\- `git commit -m "message"` – 提交并写说明

\- `git push` – 推送到 GitHub

\- `git pull` – 拉取远程更新



\### 遇到的问题

\- PowerShell 里直接输入 `algorithms/` 报错，因为这不是命令，应该用 `mkdir`

\- 记事本保存 README 乱码，解决方案：另存为 UTF-8 编码



\### 心得

\- 提交信息要用 `feat:` / `fix:` / `docs:` 等标签，便于以后回顾

\- 空文件夹需要放 `.gitkeep` 才能被 Git 追踪
