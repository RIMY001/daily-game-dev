# Markdown 常用语法速查

### 标题

在文字前加上 `#` 号即可表示标题，`#` 的数量代表标题的级别（注意 `#` 后需要加一个空格）：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### ️ 文本样式

通过特定的符号包裹文字，实现加粗、斜体等效果：

- **粗体**：`**粗体文本**` 或 `__粗体文本__`
- *斜体*：`*斜体文本*` 或 `_斜体文本_`
- ***粗斜体***：`***粗斜体文本***`
- ~~删除线~~：`~~删除线文本~~`
- 下划线：`<u>下划线文本</u>`（部分解析器支持 HTML 标签）
- ==高亮==：`==高亮文本==`（部分编辑器支持）

### 列表

Markdown 支持无序、有序以及任务列表：

- **无序列表**：使用 `-`、`*` 或 `+` 加上一个空格（支持通过缩进实现嵌套）：

```markdown
- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2
- 项目 3
```

- **有序列表**：使用数字加点 `.` 加上一个空格：

```markdown
1. 第一项
2. 第二项
3. 第三项
```

- **任务列表**：常用于待办事项：

```markdown
- [x] 已完成任务
- [ ] 未完成任务
```

### 代码

- **行内代码**：使用单个反引号包裹，例如 `print("Hello")`。
- **代码块**：使用三个反引号包裹，并可在开头指定编程语言以实现语法高亮：

```python
def hello():
    print("Hello, World!")
```

### 链接与图片

- **链接**：`[链接文字](https://www.example.com "可选的悬停标题")`
- **图片**：在链接语法前加一个感叹号 `!`，即 `

![](图片链接地址)

`

- **图片链接**（点击图片跳转）：`[

![](图片链接)

](跳转链接)`

### 引用

使用 `>` 符号来表示引用，支持多级嵌套：

```markdown
> 这是一段引用文字
> 可以有多行
>> 这是嵌套的二级引用
```

### 表格

使用竖线 `|` 分隔列，连字符 `-` 分隔表头和内容，冒号 `:` 用来控制对齐方式：

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 单元格 | 单元格   | 单元格 |
| 数据1  | 数据2    | 数据3  |
```

### 分隔线

使用三个或以上的 `-`、`*` 或 `_` 即可创建一条水平分割线：

```markdown
---
***
___
```

### 其他实用技巧

- **段落与换行**：段落之间用一个空行隔开；如果想在同一段落内换行，可以在行尾加两个空格再回车。
- **转义字符**：如果想显示 Markdown 的特殊符号（如 `*`、`#`、`[` 等），可以在符号前加上反斜杠 `\`，例如 `\*`。
- **注释**：使用 HTML 注释语法，渲染时不会显示：`<!-- 这是注释内容 -->`。
